# Pandora-University_LMS
판도라 대학의 LMS 관리 사이트

## 프로젝트 소개
Zoom API와 Youtube API를 활용한 LMS 사이트

## 개발기간
- 23년 4월 1일 ~ 23년 4월 30일 ( 1달 )

## 개발환경
- `Java 11`
- **IDE** : VSCode
- **Framework** : Springboot(2.x)
- **Database** : MySQL 8.0
- **ORM** : Mybatis

## 맡은 역할
- spring boot 사용하여 공지사항,게시판 구현
- 웹소켓을 사용하여 실시간 채팅 및 알람 구현
- Zoom API를 사용하여 Zoom 회의 개설
